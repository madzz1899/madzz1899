# Toko Permen Flutter

Website toko permen dan manisan dengan tampilan modern dan menarik, dibangun menggunakan Flutter.

## Fitur

- 5 Halaman Utama (Home, Produk, Kategori, Keranjang, Profil)
- Tampilan modern dengan animasi
- Gambar produk diambil langsung dari internet (Unsplash API)
- Sistem keranjang belanja
- Detail produk dengan modal bottom sheet
- Pencarian produk
- Kategori produk
- Halaman profil dengan menu lengkap

## Teknologi yang Digunakan

- Flutter 3.x
- cached_network_image: Untuk caching gambar dari internet
- google_fonts: Untuk typography yang menarik
- url_launcher: Untuk menghubungi kontak support
- provider: Untuk state management (opsional)

## Instalasi

1. Clone repository
2. Jalankan `flutter pub get`
3. Jalankan `flutter run`
